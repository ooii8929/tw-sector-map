## 📖 用法說明

### 搭配資金指標判讀

| 訊號組合 | 意義 | 操作 |
|---------|------|------|
| 外資連買 + 投信同向 | 法人共識看好 → 族群主升段 | 族群內選動量最強股進場 |
| 外資買 + 投信賣 | 外資換手，投信獲利了結 | 觀望，等方向明確 |
| 外資賣 + 投信買 | 投信接刀，外資出貨 | ⚠️ 小心，通常投信接不住 |
| 三大法人同步賣超 | 族群資金撤退 | 避開或減碼 |
| 累計由負轉正 | 資金回流訊號 | 關注族群內領漲股 |
| 單日爆量買超（>5日均量2x） | 異常資金湧入 | 短線族群輪動機會 |

### 搭配策略

- **L1+動量Top20**：資金流向表確認動量方向，避免逆勢持股
- **族群輪動**：觀察資金從哪個族群流出、流入哪個族群
- **進場時機**：族群資金連續 3 天淨流入 + 個股突破 → 進場訊號
- **出場警示**：持股族群資金連續 3 天淨流出 → 減碼警示
